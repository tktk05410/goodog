# 项目截图目录

此目录用于存放 README 中引用的项目截图。

## 文件对应关系

| 说明 | 文件名 |
|------|--------|
| 首页预览 | `home.png` |
| 商品详情页 | `product-detail.png` |
| 发布商品页 | `publish.png` |
| 个人中心页 | `profile.png` |
| 统计页面 | `stats.png` |
| 对话页面 | `messages.png` |
| 项目管理后台 | `admin.png` |

## 使用方式

1. 运行项目并截取对应页面。
2. 将截图保存为上方表格中的文件名。
3. 提交到 GitHub 后，README 中的图片占位符会自动显示。

> 支持格式：`png`、`jpg`、`jpeg`。若使用其他格式，请同步修改 `README.md` 中的图片路径后缀。
